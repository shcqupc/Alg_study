# 前置依赖和安装
- numpy
- scipy
- sympy
- matplotlib
- jupyter
- pyecharts

对于安装Anaconda最新版本的用户仅需安装pyecharts。

安装方法，在终端中输入：
> pip install pyecharts

如果国内网速较慢可以使用清华源：
> pip install -i https://pypi.tuna.tsinghua.edu.cn/simple pyecharts

# 使用方法
在终端中（或powershell）中输入：
> jupyter-notebook
